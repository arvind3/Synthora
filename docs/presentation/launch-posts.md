### LinkedIn Post 1
We just launched **Synthora**.

It turns the Faker library into a browser-based product for generating realistic synthetic data without setup, signup, or backend infrastructure.

Why we built it:
- teams still waste too much time handcrafting fake test data
- demo environments need believable records
- real data should not be copied into lower environments

What it does:
- explore Faker methods through a clean UI
- build structured datasets in the browser
- export JSON or CSV instantly
- stay fully client-side

Try it here: https://arvind3.github.io/Synthora/
